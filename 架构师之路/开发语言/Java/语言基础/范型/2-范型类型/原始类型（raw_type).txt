====== 原始类型（raw type) ======
创建日期 星期三 12 十二月 2018

原始类型是指不带任何类型参数的 范型类型。
比如类定义如下：

public class Box<T> {
	public void set(T t) { /* ... */ }
	// ...
}
但用Box   rawBox = new Box() 就创建 了一个Box<T>的原始类型。
这个决定于你Box类是否定义为一个带参数的类型。

原始类型出现的场景主要是java5以下的jdk中，为了兼容性，你可以将一个范型类型的实例赋值给一个原始类型，但是反过来就会在编译时收到警告。
另外，原始类型的对象在调用相应范型类型中定义的方法时也会收到警告，这个就是提醒程序员要修改它，这是不类型不安全的。

===== 不被检查的的错误消息 =====
如上所述，编译时会提示有类型不安全的情况，但不详细
可以用-Xlint:unchecked   来得到更详细的信息。
