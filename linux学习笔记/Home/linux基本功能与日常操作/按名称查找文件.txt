====== 按名称查找文件 ======
Created 星期五 27 四月 2018

===== 只查命令文件 =====
which

===== 在特定的目录中关键字查 =====
whereis  

===== 从文件索引数据库中找 =====

locate
需要定时地updatedb

以上查询都 较快，但不能查找所有 文件
时效性有影响 

===== 万能查找 =====
find

因为量大，查的较慢。但啥都能查。
