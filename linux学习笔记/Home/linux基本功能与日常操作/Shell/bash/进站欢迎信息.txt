====== 进站欢迎信息 ======
Created 星期五 01 六月 2018

[[/etc/issue]]

issue 内的各代码意义
\d 本地端时间的日期;
\l 显示第几个终端机接口;
\m 显示硬件的等级 ( i386/i486/i586/i686... );
\n 显示主机的网络名称;
\O 显示 domain name ;
\r 操作系统的版本 (相当于 uname -r )
\t 显示本地端时间的时间;
\S 操作系统的名称;
\v 操作系统的版本。

[[/etc/issue.net]] 专门针对telnet过来的用户展示

[[/etc/motd]]
这个里边可以放公共信息
