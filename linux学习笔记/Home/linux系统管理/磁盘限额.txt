====== 磁盘限额 ======
Created 星期五 06 七月 2018

===== 概念 =====

==== quota ====

1. 用来限制一个用户或组的占用磁盘的最大空间.
2. 具体配置与文件系统的不同而不同.
3. ext类的文件系统,只能在全文件系统级别控制,而且需要专门的程序
4. xfs因为在文件系统内部支持,无需专门程序, 且可以细化到某一个目录的控制
5. 只对一般用户生效, root不受控制
6. 内核要支持
7. 如果用了SELinux, 它会有更细的控制,可能只能对/home 下的目录进行限制.

==== quota的可控制的项目 ====

1. 能控制用户,组, 目录所占的空间
2. 容量限制或文件数量限制 （block 或 inode）
3. 柔性劝导与硬性规定 （soft/hard）
4. 会倒数计时的宽限时间 （grace time）
